Predicate Details:

  • map/3

    Form:
    map(Function, List, NewList)
    Description:
    NewList is the result of applying Function to every element of List. In Haskell notation: map :: (a -> b) -> [a] -> [b]

  • zip/3

    Form:
    zip(List1, List2, NewList)
    Description:
    NewList is the result of an element-wise pairing of the elements in List1 and List2. Each element of NewList is a pair, where the left element is an element from List1 and the right element is an element from List2.
    Constraints:
    List1 and List2 must have the same length.

  • zip/4

    Form:
    zip(Function, List1, List2, NewList)
    Description:
    NewList is the result of applying Function to the i-th element of both List1 and List2. In Haskell notation: zip_with :: (a -> b -> c) -> [a] -> [b] -> [c]
    Constraints:
    List1 and List2 must have the same length.
